# Drum Kit

## Skills

`DOM API` `keyboard events` `mouse events` `HTMLAudioElement` `data-* attributes` `CSS transitions`

## Task Description

**Drum Kit** is a small musical app: pressing a key on the keyboard plays a unique sound mapped to that key. Sound playback is a foundational piece of games, audio players, music apps, and audible notifications on websites.

![Drum Kit screenshot](images/js30-1.jpg)

- [Reference demo](https://irinainina.github.io/JavaScript30-1/01%20-%20JavaScript%20Drum%20Kit/index-FINISHED.html)
- [Author video walkthrough](https://youtu.be/VuN8qwZoego) (19:38)
- Parent task and scoring: [js30.md](js30.md)

## Mandatory Additional Feature

Allow the app to play sounds **both** on keyboard keypress **and** on mouse click on the corresponding on-screen pad. The visual "pressed" feedback should look the same regardless of input source. [Reference demo](https://js3001.github.io/)

## Optional Improvements

Each well-executed item below is worth **+10** points (Stage 3 in [js30.md](js30.md), capped at 30 per widget). You may also invent your own improvements of comparable complexity.

- Multiple instruments - switching the instrument swaps the set of sounds played by the same pads.
- A "play a predefined melody" mode with animated key highlights timed to the playback.
- Polished pad animation on press, e.g. a Material-style ripple.
- Floating hearts / notes that spawn on click.
- A "light show" where each note paints an abstract visual pattern on a backdrop.
- A full virtual piano with octaves and a wider keyboard mapping.
- Dark / light theme toggle that persists across reloads.
