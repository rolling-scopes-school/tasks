5. (+2) Доска (ссылка + скриншот) и Meeting Notes (ссылки на 3+ записей)
6. (+2) Лучшие PR — 3-4 PR с содержательным code review

- **Мерж дневника** — все записи замержены в `main`. Создан файл `username-merged.md` в `/development-notes/github-username/`.

> **Все три защиты обязательны.** Без командной защиты, peer review и менторской сессии — 0 баллов за соответствующий Score.
