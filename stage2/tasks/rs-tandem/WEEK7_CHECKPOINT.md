# Tandem: Week 7 Checkpoint

**Дедлайн:** 7 апреля 2026, 23:59 UTC

> Неделя подготовки к защитам. Код уже должен быть готов — эта неделя для оформления и регистрации.

## Командная часть

Выполняется один раз на команду.

1. **README-артефакты** — в `README.md` проекта в ветке `main` присутствуют все 9 пунктов:
   - О проекте (название, описание)
   - Команда (участники со ссылками на дневники)
   - Доска (ссылка + скриншот)
   - Лучшие PR (3-4 PR с содержательным ревью)
   - Meeting Notes (ссылки на 3+ записей)
   - Деплой (ссылка или инструкция)
   - Feature List (3-5 фич с шагами)
   - Чем гордимся (2-10 предложений)
   - Демо-видео (до 8 мин)

2. **Демо-видео (до 8 мин)** — запись основного user path приложения. Каждый участник может записать свою часть, команда склеивает в одно видео. Ссылка в README в ветке `main`.

## Личная часть

3. **Self-assessment** — PR в репозитории команды с файлом `self-assessment.md` в `/development-notes/github-username/`. Замержить в `main`, в файле оставить ссылку на PR (место для вопросов). Каждая фича — со ссылкой на файл или PR, ★ у 2-3 ключевых фич для демо. Подробности: [SCORE_PERSONAL.md](./SCORE_PERSONAL.md#self-assessment).

4. **Мерж дневника** — все записи замержены в `main`. Создан файл `username-merged.md` в `/development-notes/github-username/`.

5. **Регистрация на Peer Review** — заполнена Google-форма для участия в peer review (ссылка в Discord).

> **Все три защиты обязательны.** Без командной защиты, peer review и менторской сессии — 0 баллов за соответствующий Score.
