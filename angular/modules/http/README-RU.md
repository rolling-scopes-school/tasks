# [Angular schedule.](../../README-RU.md) Http
## В данном модуле студентам необходимо

- Модуль `HttpClientModule`.
- Объект `HttpTypeEvent`.
- Перехват HTTP-запросов с помощью `Http Interceptor`.

теория - 1 часа
задание - 1 часа
