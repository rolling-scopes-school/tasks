# Find a Mentor & Team

## Prerequisites

Before applying for a mentor, you must have completed the **[Angular Intro Task](../angular-intro-task/README.md)**. Applications without a completed intro task will not be considered.

---

## How to Apply for a Mentor

Post your application in the Discord channel **#students-looking-for-mentor**:
[Open channel](https://discordapp.com/channels/788026364372320256/1163363732672483369)

### Application format

```text
Name:
Location (Country/City):
Communication languages:
RS App profile link:
GitHub profile link:
Telegram profile link:
Intro Task link:
About me:
```

The mentor will reach out to you directly via Discord or Telegram.
