# [Stage#1](../../) JavaScript basics: Variable

- Learn all about variables

  - usage
  - hoisting

## Approximate learning time

45 min

## Theory

- [detailed article](https://javascript.info/variables) (~15 min)

- [hoisting](https://www.programiz.com/javascript/hoisting) (~ 10 min)
