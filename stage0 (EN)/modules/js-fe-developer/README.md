# [Stage#0.](../../) Introduction to the "JS/Front-end Developer" Profession

## In this module students need to:

- Get an understanding of the JS/Front-end developer profession
- Get an understanding of languages, technologies and tools of a JS/Front-end developer
- Familiarize yourself with the responsibilities of a JS/Front-end developer
- Familiarize yourself with requirements for a beginner JS/Front-end developer

## Approximate module completion time

2 hours
