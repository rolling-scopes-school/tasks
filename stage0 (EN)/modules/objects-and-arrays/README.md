# [Stage#0.](../../) Objects and Arrays in JavaScript. Basics

## In this module students need to study

- Objects
  - purpose of object data type, creating objects
  - object literals and properties
  - using square brackets
  - loops for working with objects
  - comparison, copying and cloning objects
- Arrays
  - purpose of array data type
  - operations with arrays
  - methods for iterating over array elements
  - searching, filtering, sorting elements in an array

## Approximate module completion time

10 hours
