# [Stage#0.](../../) Functions in JavaScript

## In this module students need to:

Reinforce:

- purpose and declaration of functions
- function calls
- variable scope in functions
- external variables in functions
- function arguments and their default values
- passing arguments by value and by reference
- returning values from functions
- choosing function names
- differences between function expression and function declaration
- arrow functions, differences between arrow and regular functions
- function as an object

Study:

- Recursion
