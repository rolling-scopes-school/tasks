# Module Topic: Introduction to JavaScript Basics

## Module Content:

1. JS Basics: statements, arithmetic, strings, syntax, variables.
2. Data types and type conversion.
3. Operators (comparison and logical).
4. Working with loops.
5. Conditional constructs, boolean variables and expressions.
6. Functions in JS: declaration, invocation, properties, methods.
