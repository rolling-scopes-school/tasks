# [Stage#0.](../../) CSS Basics

## In this module students need to:

Get familiar with CSS basics:

- connecting styles
- selectors (specificity, cascade, inheritance)
- text styles, fonts
- units of measurement
- background, images in CSS
- box model

## Approximate module completion time

9 hours

## Theory

1. You need to complete the interactive course https://ru.code-basics.com/languages/css (8 hours)
2. Watch the lecture recording on [HTML and CSS basics](https://youtu.be/4Zj7tRlQJao?list=PLzLiprpVuH8df24MzZp-l5QMsJWJbi9qP)

## Practice

1. You need to pass the test "Test CSS Basics [RU]" in RS APP > Auto Test
