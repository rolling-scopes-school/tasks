# [Stage#0.](../../) HTML Basics

## In this module students need to:

Get familiar with HTML basics:

1. Syntax
2. Text elements (paragraphs, links, lists, tables)
3. Media elements (images)
4. HTML document structure (semantics)
5. Working with forms (HTML only)

## Approximate module completion time

12 hours

## Theory

1. You need to complete the interactive course https://ru.code-basics.com/languages/html (11 hours)
2. [Watch the lecture recording on HTML and CSS basics](https://youtu.be/4Zj7tRlQJao?list=PLzLiprpVuH8df24MzZp-l5QMsJWJbi9qP)

## Practice

1. You need to pass the test "Test HTML Basics [RU]" in RS APP > Auto Test
