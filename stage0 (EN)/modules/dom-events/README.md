# [Stage#0.](../../) DOM Events

## In this module students need to study:

1. Event. Definition.
2. Event-based interfaces. Complete list from MDN.
3. Which events to start with?
4. Event handlers
   Ways to bind a handler to a specific event:
   4.1. In the DOM tree with on<event> attribute
   4.2. Assign property to DOM object
   4.3. Register handler on target
   4.4. Exceptions
5. Event object
6. Bubbling and capturing
7. Delegation
8. Default actions
9. Generating custom events

## Approximate module completion time

4 hours
