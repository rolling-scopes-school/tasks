# Module "Data Processing CLI"

## In this module, students study:

- REPL architecture for Node.js CLI applications
- Working with process arguments and interactive commands
- File system navigation and path resolution
- Streams API for efficient data processing
- Compression, hashing, and file transformation pipelines

## Estimated time to complete the module

20 hours
