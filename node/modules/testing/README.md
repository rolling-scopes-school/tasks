# Module "Testing basics"

## In this module, students study:

- Testing pyramid
- FIRST principles
- Jest basics
  - Basic using of matchers
  - Table tests
  - Testing of class
  - Testing asynchronous code
  - Mocking
  - Partial mocking

## Estimated time to complete the module

5 hours
