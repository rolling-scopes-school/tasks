# Module "REST Service: Testing"

## In this module, students study:

- Unit testing strategy for backend applications
- Testing Nest.js services in isolation
- Mocking dependencies and side effects
- Testing guards and authentication flows
- Maintaining quality in a growing API codebase

## Estimated time to complete the module

10 hours
