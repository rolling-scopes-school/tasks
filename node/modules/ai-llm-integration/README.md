# Module "Knowledge Hub.AI: LLM Integration"

## In this module, students study:

- Integrating an external LLM API into a Nest.js backend
- Prompt design for business features
- Building AI endpoints over existing domain data
- Handling API keys and external provider failures
- Structuring AI integration in service architecture

## Estimated time to complete the module

15 hours
