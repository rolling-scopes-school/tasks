# Module "Knowledge Hub.AI: RAG & Vector DB"

## In this module, students study:

- Retrieval-Augmented Generation (RAG) architecture
- Embeddings and semantic search basics
- Vector database setup and integration
- Building grounded answer pipelines with source attribution
- Operating AI, PostgreSQL, and vector DB in one Docker environment
